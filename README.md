# Vue
Vue router app sample

## Install Vue dev
Install development tools

### Install nodejs, npm
```bash
# install npm, nodejs
apt install npm, nodejs

# install vue cli
npm i -g @vue/cli
```

### Create vue app
```bash
vue create app-name
```

### Add router (history mode)
```bash
# install router
npm install vue-router

# add to vue app
cd app-name

# add router
vue add router
```

### Run local server
```bash
# goto app folder
cd app-name

# run in background
npm run server &
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
